# Local Gemma 4 Code Chat

## 1) Download the local model

```bash
ollama pull gemma4:31b
```

## 2) Start Ollama

```bash
ollama serve
```

## 3) Start the chat server

```bash
cd /Users/politrons/development/GenAICode
python3 gen_code_server.py
```

## 4) Open the chat

Open:

- `http://127.0.0.1:8787`
